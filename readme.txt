Sistema de Gestión para Estudiantes de Preescolar

Sistema desarrollado en **Visual Basic 2010** utilizando una base de datos en **Microsoft Access**. Está diseñado para facilitar el registro y seguimiento de grados escolares, profesores y estudiantes en una institución preescolar.

## Estructura del Repositorio

- `/sistema de gestion de estudiantes/`: Contiene todo el código del proyecto 
- `sistema de gestion.zip`: Versión compilada del proyecto. Incluye el ejecutable y la base de datos lista para probarse.

## Funcionalidades principales

- Registro, edición y eliminación de estudiantes, profesores, grados escolares y usuarios.
- Almacenamiento en base de datos Access.
- Reportes básicos.

## ¿Cómo probarlo?

1. Descargar el archivo [`sistema de gestion.zip`] y extraerlo.
2. Abrir la carpeta extraída.
3. Ejecutar el archivo .exe
4. Asegúrate de tener:
   - Visual Basic 2010 
   - Windows con .NET Framework compatible con Visual Basic 2010
   - Microsoft Access o Access Database Engine instalado

## ¿Cómo abrir el código fuente?

Abrir la carpeta `sistema de gestion estudiantes` y ejecutar el archivo `.vbproj` que tiene el mismo nombre. Usar **Visual Basic 2010** o en versiones compatibles.

Preschool Management System

This system was developed in **Visual Basic 2010**, using a database with Microsoft Access. It is developed to make easier records and keeps track of grade levels, teachers and students in a preschool

## Repository structure

- `/sistema de gestion de estudiantes/`: It has the entire code of the project
- `sistema de gestion.zip`: Compiled version of the project. This include the executable and the database ready to run the system.

## Main features

- CRUD operations for students, teachers, grade levels and users.
- Data storage in a Microsoft Access database.
- Basic reports.

## How to run the system?

1. Download the file [`sistema de gestion.zip`] and then extract it.
2. Open the extracted folder.
3. Be sure to have:
   - Visual Basic 2010 
   - Windows with .NET Framework compatible with Visual Basic 2010
   - Microsoft Access or Access Database Engine installed

## How to view the code?

Open the folder called `sistema de gestion estudiantes` and execute the file `.vbproj` which has the same name of the folder. Use **Visual Basic 2010** or in compatible versions
